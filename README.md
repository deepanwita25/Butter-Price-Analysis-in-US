# Butter-Price-Analysis-in-US

This project is designed to show the entire process of data analysis. It has the following steps
- Researching about the topic which is Butter Price in US
- Collecting data from government sources like USDA and clal
- Extraction, transformation and loading of the data in MySQLServer
- Loading the data in Power BI through the data pipeline
- Modelling the data in Power BI Desktop
- Visualizing the data in Power BI report

# The Analysis

Butter prices increased significantly in 2024. It was expected to stay high but suddenly the prices started dropping and there was a major drop post July 2025. (View the production data in USD/kg or USD/tonne by changing selecting from the slicer)
